# 📦 재고 관리 앱 (STOCK/mgr)

모바일(아이폰) 최적화 재고 관리 웹앱입니다.  
GitHub Pages로 배포해 사용할 수 있습니다.

## 🚀 GitHub Pages 배포 방법

1. **GitHub 저장소 생성**  
   GitHub에서 새 저장소를 만드세요 (예: `stock-manager`)
1. **파일 업로드**  
   `index.html` 파일을 저장소 루트에 업로드하세요.
1. **Pages 활성화**  
   저장소 → Settings → Pages → Source: **Deploy from a branch** → Branch: `main` / `/(root)` → Save
1. **접속**  
   `https://[사용자명].github.io/[저장소명]/` 으로 접속
1. **아이폰 홈 화면에 추가** (앱처럼 사용)  
   Safari에서 접속 → 공유 버튼 → “홈 화면에 추가”

-----

## ✨ 기능

|기능         |설명                         |
|-----------|---------------------------|
|상품 추가/수정/삭제|이모지, 카테고리, 단위, 가격 등        |
|수량 빠른 조절   |목록에서 +/− 버튼으로 즉시 변경        |
|부족/품절 경고   |최소 수량 설정 → 자동 색상 표시        |
|카테고리 탭     |자동 생성되는 카테고리별 필터           |
|검색         |실시간 상품명 검색                 |
|정렬         |이름순, 수량순, 최근 수정순           |
|변경 이력      |상품별 최근 20개 수량 변경 기록        |
|재고 총액      |단가 × 수량 자동 계산              |
|CSV 내보내기   |엑셀 호환 CSV 다운로드             |
|로컬 저장      |localStorage 사용, 인터넷 없이도 동작|

-----

## 📱 아이폰 사용 팁

- Safari에서 홈 화면에 추가하면 앱처럼 전체 화면으로 사용 가능
- 데이터는 기기의 브라우저에 저장됩니다 (다른 기기와 동기화 안됨)
- CSV 내보내기로 정기 백업 권장
